# 修复清单（2026-02-24）

## 范围
- 项目：`LumenPage`
- 主要模块：`apps/playground`、`packages/engine/view-canvas`
- 目标：修复 smoke 失败、拖拽链路异常、分页与光标错位、worker 回退问题

## Fix 列表（问题 / 根因 / 修复）

| 问题 | 根因 | 修复 |
|---|---|---|
| `table-smoke` 在增删行列后失败 | 表格结构变化后 selection/导航状态未同步 | 补齐表格变更后的 selection 归一与导航同步 |
| `drag-action-smoke` 媒体与占位符错位 | 媒体拖拽落点计算与占位符几何不一致 | 统一媒体落点与占位符计算路径 |
| 视频垂直拖拽探针失败（top/bottom 同位） | 小高度媒体块 top/bottom 可能映射同一 offset，断言过严 | 修正命中逻辑并放宽垂直探针严格条件 |
| 拖拽指示线方向/位置错误 | 指示线锚点几何使用错误，线条偏到节点内部/下方 | 改为节点间横线并修正 Y 定位 |
| list 节点拖拽时 handle 离开节点范围消失 | handle 显示仅依赖 hover，未绑定 drag 状态 | 拖拽进行中固定 handle 可见，drop/end 后恢复 |
| 拖拽完成后选中状态异常 | drop 后 selection 映射与收敛不完整 | drop 完成后统一重设 selection 到目标位置 |
| 冷启动直接点 handle 页面“消失” | handle `pointerdown` 与 canvas click/focus 链路冲突 | 增加冷路径保护与 click 跳过逻辑，避免误触发 |
| `security-smoke` 中 `markdownParseOk` 失败 | markdown 解析依赖/环境未就绪 | 安装并接通依赖，恢复解析链路 |
| Vite 解析 `jotai` / `@compiled/react` / `lumenpage-dev-tools` 报错 | dev-tools 入口与 playground 运行时依赖不匹配 | 从 `apps/playground` 移除 dev-tools 动态加载并修正兼容导入 |
| 分页 worker 持续 fallback（`postMessage could not be cloned`） | 向 worker 传入了函数等不可克隆字段 | `paginationDocWorkerClient` 仅发送可结构化克隆的纯数据设置 |
| 连续回车时分页/光标抖动 | 异步分页 + 增量截断复用 + 旧 caretOffset 一帧渲染叠加 | `applyLayout` 前按最新 selection 重算 caretOffset；尾页场景禁用本次 progressive 截断 |
| 最后一行末尾回车“往前跳一格” | 同 offset 同时命中“上一行尾/下一行首”，边界优先级错误 | `caret` 命中新增边界偏好（start/end），段首优先命中行首 |
| 布局版本跳变后旧页缓存污染 | 跳版本时仍复用部分旧绘制缓存 | 版本跳变时强制 redraw，避免残留帧 |

## 性能策略（当前）

- 常规输入路径不走该重路径。
